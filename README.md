# 어플 소개
BaobabStudyCafe 출입을 위해 제작한 비공식 어플

다음과 같은 기능을 제공합니다.
- 기간권 남은 기간 확인
- 어플에 바코드를 등록하여 손쉽게 확인
- 바탕화면에 바코드 위젯을 추가하여, 원활한 입퇴실 가능

플레이스토어에 정식 등록되지않은 어플이기 때문에
설치 직후, 보안 정책 관련 창이 자주 뜹니다.
오픈 소스 어플리케이션이기 때문에 걱정 없이 설치하셔도 됩니다.
